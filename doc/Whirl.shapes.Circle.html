Class: Circle

Whirl.shapes. Circle

Represents a circle defined by a center coordinate and a radius.

Can be used for things such as physics with circular objects, radius bound checking, distance calculation, etc.

Circles are constructed using the Whirl.shapes.Circle factory method, but the underlying class can be accessed with Whirl.shapes.Circle._class.

new Circle( [x] [, y] [, r])

Parameters:
Name Type Argument Default Description
x number | Whirl.shapes.Point <optional>
 0

X-coordinate of the center-point. If giving an instance of a Point then the first argument should be the Point instance, and the second argument should be the radius, instead.
y number <optional>
 0

Y-coordinate of the center-point. If the first argument is a Point instance then this argument represents the radius, instead.
r number <optional>
 0

Radius of the circle.
Examples
Whirl.shapes.Circle(25, 25, 50); // Circle {x: 25, y: 25, r: 50}
Whirl.shapes.Circle(
	Whirl.shapes.Point(25, 25),
	50,
); // Circle {x: 25, y: 25, r: 50}

Determine if a point is inside this circle.

Parameters:
Name Type Argument Description
px number | Whirl.shapes.Point

X-coordinate of the point. An instance of a Point can also be given instead as the only argument to determine if it is inside this circle.
py number <optional>

Y-coordinate of the point.
Returns:
Type
boolean
Examples
Whirl.shapes.Circle(25, 25, 50).isPointInside(30, 20); // true
const circ = Whirl.shapes.Circle(25, 25, 50);
const point = Whirl.shapes.Point(30, 20);

circ.isPointInside(point); // true
